开始写 CoffeeScript 啦，如何开始？从哪儿学习？

## 安装

### CoffeeScript 环境

```bash
npm install -g coffee-script
```

### Sublime Text 插件

> Sublime Text 竟然默认没有.coffee 文件的语法高亮支持

推荐 [Better CoffeeScript](https://packagecontrol.io/packages/Better%20CoffeeScript) 插件
